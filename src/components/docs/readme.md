## 底部按钮栏 params:type
1. 从所有带有底部状态栏的页面分离
  1. tabbar  -- 底部导航  type:'tabbar'
  2. button  -- 单独确定按钮  type:'btn'
    1. 文本内容 -- 参数：content
  3. context -- 评论按钮栏 type:'context'
  4. pay -- 付款类 type:'pay'
    1. btn-text -- 按钮
    2. content  -- 内容
  5. order -- 订单类 type:'order'

## 列表分离

2. 顶部无 -- 底部无
3. 顶部用户详情 -- 底部按钮栏
